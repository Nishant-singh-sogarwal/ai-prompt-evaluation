# Prompt Refinement 2: Email Tone

## Original Prompt
Write an email.

## Original Response
The model produced a very generic email without clear purpose or tone.

## Improved Prompt
Write a formal email to my project mentor requesting a meeting tomorrow at 4 PM to discuss project progress. Keep it under 120 words.

## Improved Response
Subject: Request for Meeting on Project Progress

Dear Mentor,

I hope you are doing well. I would like to request a short meeting tomorrow at 4:00 PM to discuss my current project progress and next steps.

Please let me know if this time is convenient for you. If not, I can adjust as per your availability.

Thank you.

Best regards,
[Your Name]

## Why Improved Prompt Gives Better Output
The improved prompt clearly defines audience, purpose, time, tone, and length. This reduces ambiguity and produces a useful response.
