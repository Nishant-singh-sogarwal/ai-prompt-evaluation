# Prompt Refinement 02: Formal Email

## Original Prompt
Write an email.

## Original Response
The model generated a generic email without clear context, tone, or purpose.

## Improved Prompt
Write a formal email to a project mentor requesting a meeting tomorrow at 4 PM to discuss project progress. Keep it under 120 words.

## Improved Response
Subject: Request for Meeting on Project Progress

Dear Mentor,

I hope you are doing well. I would like to request a short meeting tomorrow at 4:00 PM to discuss my project progress and next steps.

Please let me know if this time works for you. If needed, I can adjust as per your availability.

Thank you.

Best regards,
[Your Name]

## Why the Improved Prompt Gives Better Output
The improved prompt defines audience, purpose, time, tone, and word limit. This helps the model generate a focused and practical response.
