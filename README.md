# AI Prompt Evaluation Portfolio

## Project Overview
This repository is a beginner-level portfolio project focused on **Prompt Engineering** and **AI Response Evaluation**. It compares responses from **ChatGPT** and **Google Gemini** on common tasks to understand output quality, reliability, and usefulness.

The work is presented from the perspective of a **Computer Science fresher** learning how to write better prompts and evaluate AI outputs in a structured way.

## Objective
- Practice writing clear prompts for different use cases.
- Compare responses from two popular AI models.
- Evaluate outputs using a simple rubric.
- Identify hallucinations in factual questions.
- Build an interview-ready repository that shows practical AI evaluation skills.

## Skills Demonstrated
- Prompt writing and refinement
- Side-by-side response comparison
- Basic AI response quality evaluation
- Hallucination detection and factual verification
- Structured documentation in Markdown

## Evaluation Criteria
Responses are evaluated on:
- Accuracy
- Clarity
- Completeness
- Relevance
- Instruction Following
- Reasoning
- Overall Quality

A detailed scoring rubric is available in:
- `evaluation-rubrics/response-evaluation-rubric.md`

## Learning Outcomes
Through this project, I learned:
- Small prompt changes can significantly improve output quality.
- Different models can produce different levels of detail and correctness.
- Hallucinations can happen even with confident answers.
- A rubric helps in making fair and repeatable comparisons.
