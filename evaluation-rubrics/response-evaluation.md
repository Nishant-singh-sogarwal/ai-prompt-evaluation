# Response Evaluation Rubric

Use this rubric to score AI responses on a beginner-friendly 1–5 scale.

## Scoring Scale
| Score | Meaning |
|---|---|
| 1 | Poor |
| 2 | Below Average |
| 3 | Average |
| 4 | Good |
| 5 | Excellent |

## 1) Accuracy
| Score | Description |
|---|---|
| 1 | Mostly incorrect facts or logic |
| 2 | Multiple errors |
| 3 | Mostly correct with minor mistakes |
| 4 | Correct with very small issues |
| 5 | Fully correct and reliable |

## 2) Clarity
| Score | Description |
|---|---|
| 1 | Very confusing |
| 2 | Hard to read |
| 3 | Understandable but basic |
| 4 | Clear and readable |
| 5 | Very clear and easy to follow |

## 3) Completeness
| Score | Description |
|---|---|
| 1 | Misses most required points |
| 2 | Misses important details |
| 3 | Covers main request |
| 4 | Covers most details well |
| 5 | Fully addresses all requirements |

## 4) Relevance
| Score | Description |
|---|---|
| 1 | Off-topic |
| 2 | Partially relevant |
| 3 | Mostly relevant |
| 4 | Highly relevant |
| 5 | Completely on-topic |

## 5) Instruction Following
| Score | Description |
|---|---|
| 1 | Ignores instructions |
| 2 | Follows few instructions |
| 3 | Follows main instructions |
| 4 | Follows nearly all instructions |
| 5 | Follows all instructions exactly |

## 6) Reasoning
| Score | Description |
|---|---|
| 1 | Illogical |
| 2 | Weak logic |
| 3 | Basic logic |
| 4 | Good logical flow |
| 5 | Strong, consistent reasoning |

## 7) Overall Quality
| Score | Description |
|---|---|
| 1 | Not useful |
| 2 | Limited usefulness |
| 3 | Acceptable |
| 4 | Useful and dependable |
| 5 | Excellent and interview-ready |

## Suggested Usage
1. Score each criterion from 1 to 5.
2. Add one short note for strengths and one for improvements.
3. Calculate average score for final model comparison.
