# Response Evaluation Rubric

Use this rubric to score AI responses consistently on a 1–5 scale.

## Scoring Scale
| Score | Meaning |
|---|---|
| 1 | Poor |
| 2 | Below Average |
| 3 | Average |
| 4 | Good |
| 5 | Excellent |

## 1) Accuracy
| Score | Description |
|---|---|
| 1 | Mostly incorrect facts or logic |
| 2 | Several factual/logical errors |
| 3 | Mostly correct with minor errors |
| 4 | Correct with very small issues |
| 5 | Fully correct and reliable |

## 2) Clarity
| Score | Description |
|---|---|
| 1 | Very confusing |
| 2 | Hard to follow |
| 3 | Understandable but not smooth |
| 4 | Clear and readable |
| 5 | Very clear and easy to understand |

## 3) Completeness
| Score | Description |
|---|---|
| 1 | Misses most required parts |
| 2 | Misses important details |
| 3 | Covers basic request |
| 4 | Covers most details well |
| 5 | Fully addresses all requested parts |

## 4) Relevance
| Score | Description |
|---|---|
| 1 | Mostly off-topic |
| 2 | Partially relevant |
| 3 | Generally relevant |
| 4 | Highly relevant |
| 5 | Fully focused on prompt |

## 5) Instruction Following
| Score | Description |
|---|---|
| 1 | Ignores instructions |
| 2 | Follows few instructions |
| 3 | Follows main instructions |
| 4 | Follows nearly all instructions |
| 5 | Follows all instructions exactly |

## 6) Reasoning
| Score | Description |
|---|---|
| 1 | Illogical or unsupported |
| 2 | Weak reasoning |
| 3 | Basic reasoning |
| 4 | Good reasoning with clear logic |
| 5 | Strong, consistent, and well-supported reasoning |

## 7) Overall Quality
| Score | Description |
|---|---|
| 1 | Not useful |
| 2 | Limited usefulness |
| 3 | Acceptable |
| 4 | Useful and dependable |
| 5 | Excellent and interview-ready |

## Suggested Use
1. Score each criterion from 1 to 5.
2. Add short notes for strengths and weaknesses.
3. Calculate average score for final comparison.
