# Prompt Comparison 05: General Reasoning Task

## Prompt
If all roses are flowers and some flowers fade quickly, can we conclude that all roses fade quickly? Explain briefly.

## ChatGPT Response
No, we cannot conclude that all roses fade quickly. The statement says only some flowers fade quickly, not all flowers. So there is not enough information to say all roses fade quickly.

## Google Gemini Response
No. We only know that some flowers fade quickly, but that does not mean every flower (or every rose) is in that group.

## Comparison
Both responses are logically correct. ChatGPT explains with simpler wording. Gemini is concise and formal.
