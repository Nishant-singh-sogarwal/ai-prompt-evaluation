# Prompt Comparison 5: General Reasoning

## Prompt
If all roses are flowers and some flowers fade quickly, can we say all roses fade quickly? Explain briefly.

## ChatGPT Response
No, we cannot say that all roses fade quickly. The statement only says that *some* flowers fade quickly, not all flowers. Roses are flowers, but we do not have enough information to conclude that all roses fade quickly.

## Google Gemini Response
No. From the given statements, we only know that some members of the flower group fade quickly. This does not logically imply that every rose belongs to that subset.

## Comparison
- Both responses are logically correct.
- ChatGPT explains in simpler language.
- Gemini uses more formal logic wording.
