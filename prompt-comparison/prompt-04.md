# Prompt Comparison 04: Summarization Task

## Prompt
Summarize the text in 3 bullet points:
"Machine learning is a branch of AI that learns from data. It is used in recommendation systems, spam detection, and image recognition. Better data quality improves model performance."

## ChatGPT Response
- Machine learning is a part of AI where systems learn from data.
- It is used in recommendations, spam filtering, and image recognition.
- Better quality data usually leads to better model performance.

## Google Gemini Response
- Machine learning allows computers to identify patterns from data.
- Typical use cases include recommendation engines, spam detection, and image recognition.
- Model accuracy often improves when training data quality is high.

## Comparison
Both summaries are accurate and concise. ChatGPT is simpler in wording. Gemini gives slightly richer vocabulary.
