# Prompt Comparison 03: Email Writing Task

## Prompt
Write a polite email to a professor requesting a 2-day deadline extension for an assignment.

## ChatGPT Response
Subject: Request for Assignment Deadline Extension

Dear Professor,

I hope you are doing well. I am writing to request a 2-day extension for the assignment due this Friday, as I have been unwell.

I would be grateful if I could submit it by Sunday. I apologize for any inconvenience.

Thank you for your understanding.

Sincerely,
Student Name

## Google Gemini Response
Subject: Extension Request for Assignment

Respected Professor,

I request a 2-day extension for the assignment deadline due to health reasons. I will submit it by Sunday.

Thank you for your consideration.

Regards,
Student Name

## Comparison
Both responses are polite and relevant. ChatGPT includes better tone and structure. Gemini is concise and direct, but with less detail.
